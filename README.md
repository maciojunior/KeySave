💻 Como executar o Frontend
Clone esse repositório:

$ git clone https://github.com/maciojunior/KeySave.git

Entre no diretório do projeto:

cd KeySave

Instale as dependências do projeto:

$ npm install

Instale as dependências do projeto:

$ npx expo start

Acessar web-mode:

pressione w
